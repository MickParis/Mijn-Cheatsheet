Welkom op mijn Cheatsheet

Op deze website vind je een cheatsheet met handige code snippets die je kunt gebruiken bij het ontwikkelen van websites.

Categorie Code Uitleg
HTML <!DOCTYPE html> Deze code geeft aan dat het een HTML document is.
HTML <html lang="en"> Deze code geeft aan dat de taal van het document Engels is.
HTML <head> Deze code bevat informatie over het document.
HTML <body> Deze code bevat de inhoud van het document.
HTML <h1>Titel</h1> Deze code maakt een kop 1 aan.
HTML <h2>Titel</h2> Deze code maakt een kop 2 aan.
HTML <h3>Titel</h3> Deze code maakt een kop 3 aan.
HTML <h4>Titel</h4> Deze code maakt een kop 4 aan.
HTML <h5>Titel</h5> Deze code maakt een kop 5 aan.
HTML <h6>Titel</h6> Deze code maakt een kop 6 aan.


Categorie Code Uitleg
CSS body {
margin: 0;
padding: 0;
}		 Deze code zorgt ervoor dat de body geen margin en padding heeft.
CSS body {
font-family: Arial
}		 Deze code zorgt ervoor dat de body het lettertype Monospace heeft.
CSS font-style: bold; Deze code bepaald de style van het lettertype.
CSS header .background {
margin: 0px;
}		 Deze code bepaald de totale marge van de achtergrond.
CSS header .background {
padding: 0px;
}		 Deze code bepaald de totale padding van de achtergrond.
CSS header .background {
background-color: #f1f1f1;
}		 Deze code bepaald de achtergrondkleur van de achtergrond.
CSS header .background {
background-image: url('background.jpg');
}		 Deze code bepaald de achtergrondafbeelding van de achtergrond.
